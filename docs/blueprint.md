# **App Name**: Addis Crown Legal Assistant

## Core Features:

- Court & Case Header Input: Allows users to input court level, bench/division, city, file number, and date to dynamically generate the case header.
- Party Information Input: Enables users to add and manage applicant and respondent details including name, ID number, phone, and address.
- Legal Fact Auto-Injection: Uses pre-defined facts and legal arguments to suggest appropriate wording.
- Smart Evidence Linking: Leverages identified facts to automatically link and display related evidence, which the LLM will determine when appropriate, serving as a tool to save lawyers' time and effort in collecting evidence for litigation
- Maintenance Calculator: Offers a tool to compute estimated monthly payments per child, enhancing pleading clarity and preciseness, allowing to focus the work on the details.
- Document Generation and Printing: Generates a formatted legal document based on user inputs, optimized for printing and legal submission.

## Style Guidelines:

- Primary color: Crown dark (#0D1B2A) for a professional, serious feel. This choice conveys reliability, security, and stability.
- Background color: Light gray (#F3F4F6) to provide a clean, neutral backdrop that allows the content to stand out.
- Accent color: Crown accent (#3CC56D), a vivid, analogous green, is used to highlight key interactive elements and CTAs, subtly evoking concepts of growth and progress.
- Body and headline font: 'Noto Sans Ethiopic' sans-serif for Amharic text, paired with 'Inter' sans-serif for latin characters in the UI for clarity.
- Use simple, clear icons related to legal processes, document types, and user actions, such as a scale for justice, a pen for signing, or a file for documents.
- Implement a split-screen layout with a sidebar for navigation, an editor column for input, and a preview column for document display.
- Incorporate subtle transitions and animations for UI elements, such as sliding panels or fading effects, to provide a smooth and engaging user experience.
